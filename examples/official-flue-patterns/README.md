# Official Flue patterns included here

These examples are adapted from the official Flue repo:

https://github.com/withastro/flue

They are intentionally kept outside root `agents/` so they do not change the deployed Worker until copied into place.

## Examples

- `support-r2/`: R2-backed virtual sandbox knowledge-base agent.
- `cloudflare-sandbox/`: Cloudflare Sandbox container-backed assistant.

Read `../../OFFICIAL_FLUE_EXAMPLES.md` for the full walkthrough, activation steps, and current local readiness notes.
